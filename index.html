■ 状態の説明

  • 広告○広告の入札価格が最安値の 25円 かつ、広告が 最上段に出ている 状態。
  • 広告×広告が 下段に出ている 状態。
  • 広告無広告を打っていない状態。
  • PRオプ○PRオプションが最安値の 7% かつ、SEO順位が 2列目以上 の状態。
    （送り込み元で上に上げておきたいものは、7%でなくても例外的に○。）
  • PRオプ×SEO順位が 3列目以下 の状態。
  • 撤退検討▲広告の採算が合わないため削除し、PRオプションのみで販売できるかを検証しているもの。
  • 撤退■在庫が無くなり次第販売終了予定のもの。

■ 撤退までの流れ

  1. まずは広告を出して売っていく
  2. 採算が合わなくなったら広告を削除して、PRオプションだけで運用
  3. それでも利益が出ない場合は撤退認定する

⚙️ GitHub連携設定

GitHub に Personal Access Token (PAT) を設定すると、Ycheckのデータが複数のPC/ブラウザ間で共有できるようになります。

PATは github_pat_... 形式の長い文字列です。
このPC（このブラウザ）のlocalStorageにのみ保存され、Claude・GitHub以外に送信されません。

保存先（現在のショップ）:
data/yukaiya/ycheck_data.json
⚠ 初回利用時の注意
GitHub上にまだデータが無い状態で「今すぐGitHub保存」を実行すると、このPCのデータがGitHubの初期データとしてアップロードされます。
他PCから先にアップロード済みの場合は、先に「GitHubから再取得」を実行してください。
